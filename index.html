ImageBot

Bot Discord écrit en Python qui envoie une image aléatoire depuis un répertoire défini quand une commande est lancée. Le répertoire où sont les images, le préfixe de la commande, le(s) nom(s) (alias) de la commande, le message qui s'envoie et le token sont configurables via un fichier JSON.
