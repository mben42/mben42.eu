McAutoWhitelist

Script pour whitelist simplement une liste de joueurs

Téléchargement

  • Exécutable Windows
  • Code source (GitHub)

Utilisation

Extraire le fichier, qui devrait créer un dossier whitelist. La liste des joueurs doit être mise dans le fichier whitelist.txt (un pseudo par ligne). La commande à taper pour whitelist doit être mise dans le fichier command.txt (par défaut : /whitelist add).

Une fois les deux fichiers remplis, exécuter l'application, puis se placer sur sa fenêtre minecraft (5 secondes de délai). Les commandes sont envoyées à 5 secondes d'intervalle jusqu'à la fin de la liste.
